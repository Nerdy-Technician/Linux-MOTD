# Linux-MOTD

🌟 This repository contains a simple Message of the Day (MOTD) for Ubuntu/Debian systems. 🌟

## Motivation

The MOTD is displayed to users when they log in to the system. It can be used to provide important information, updates, or announcements to users.

## Installation

To install the MOTD, follow these steps:

1. Clone the repository: `git clone https://github.com/Brandon-Roff/Linux-MOTD.git`
2. Navigate to the repository directory: `cd Linux-MOTD`
3. Copy the MOTD file to the appropriate location: `sudo cp motd /etc/motd`

## Usage

Once installed, the MOTD will be displayed to users when they log in to the system. You can customize the content of the MOTD by modifying the `motd` file.
